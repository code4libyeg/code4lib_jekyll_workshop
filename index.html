What is Jekyll?

Outline

  • Create GitHub repository
  • Make sure Ruby is installed

Create GitHub repository

  • Log in
  • Create new...
  • Notice the `remote master` line
  • Project repo vs. User repo
  • Make sure Ruby is installed...
  • Install Jekyll library

First jekyll command

  • jekyll new
  • Directory name should be the same as your repo name.
  • Run jekyll serve
  • Go to http://localhost:4000

Make some changes

  • _config.yml
  • index.html
  • _posts/

Commit to GitHUb

  • Add remote master
  • git add .
  • git commit -m 'initial commit'
  • git push origin [master/branch]
